# KitchenOwl

This directory contains the Kubernetes manifests for deploying KitchenOwl.

## Files

- `namespace.yaml` - Kubernetes namespace
- `deployment.yaml` - Application deployment with PVC
- `service.yaml` - Service for the application
- `ingress.yaml` - Ingress for the application
- `kustomization.yaml` - Kustomization for the application
- `README.md` - This file
