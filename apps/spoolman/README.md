# Spoolman

Filament inventory management for your 3D printers.

## Access
- Internal: [spoolman.ganam.app](http://spoolman.ganam.app)

## Configuration
- Image: `ghcr.io/donkie/spoolman:0.23.1`
- Database: SQLite (default)
- Port: 8000 (container), 80 (service)
- Storage: 1Gi PVC (Longhorn)

## Environment Variables
- `TZ`: Asia/Jerusalem
